# TypoCorrectorBase
A simple spell checker comparison test project

This project was developed to compare the Zemberek typo corrector, a BiLSTM-based model, and n-gram based models.

Used Datasets:

WikiTurkish Corpus: https://www.kaggle.com/datasets/osmankagankurnaz/turkish-wikipedia-dataset

Turkish Spelling Dictionary: https://huggingface.co/datasets/asimokby/Turkish-Spelling-Dictionary
